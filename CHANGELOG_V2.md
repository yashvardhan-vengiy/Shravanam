# Shravanam V2 Changes

## Fixed

- Lecture library now requires Google login.
- Lecture detail pages now require Google login.
- Users cannot access tracking pages without login.
- Homepage clearly says login is required for private dashboard and progress history.

## Added

- Larger starter catalog SQL file: `database/seed_rigorous_catalog.sql`
- CSV import template: `database/import_template.csv`
- Documentation explaining why only a few lectures appear initially.

## Still intentionally absent

- No audio player
- No Play button
- No audio upload
- No Supabase Storage
- No playback tracking
- No resume-time saving

Shravanam remains a Goodreads-style devotional hearing dashboard, not a Spotify-style audio platform.
